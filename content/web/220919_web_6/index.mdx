- 웹 페이지로 이동하는 링크를 표시하는 태그

- href 속성으로 이동할 페이지 주소를 지정한다.

## 속성

| 속성           | 값                                                                                                                                                    | 설명                                                                                        |
| -------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------- |
| download       | 파일 이름                                                                                                                                             | 링크 클릭시 타겟 다운로드                                                                   |
| href           | url                                                                                                                                                   | 이동할 페이지 주소                                                                          |
| hreflang       | 언어 코드                                                                                                                                             | 연결된 문서의 언어 지정                                                                     |
| media          | 미디어 쿼리                                                                                                                                           | 연결된 문서를 최적화할 미디어/기기 지정                                                     |
| ping           | url 목록                                                                                                                                              | 링크 클릭 시 브라우저가 보내는 짧은 http post요청을 전달 받을 공백으로 구분된 url 목록 지정 |
| referrerpolicy | no-referrer<br/>no-referrer-when-downgrade<br/>origin<br/>origin-when-cross-origin<br/>same-origin<br/>strict-origin-when-cross-origin<br/>unsafe-url | 링크에 전달할 참조자 정보 명시                                                              |
| rel            | alternate<br/>author<br/>bookmark<br/>external<br/>help<br/>license<br/>next<br/>nofollow<br/>noreferrer<br/>noopener<br/>prev<br/>search<br/>tag     | 현재 문서와 링크로 연결된 문서의 관계 명시                                                  |
| target         | \_blank<br/>\_parent<br/>\_self<br/>\_top                                                                                                             | 링크로 연결된 문서를 어느 위치에서 열 것인지 명시                                           |
| type           | 미디어 타입                                                                                                                                           | 연결된 문서의 미디어 타입을 명시                                                            |
